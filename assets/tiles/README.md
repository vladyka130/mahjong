# Зображення плиток маджонгу

Поклади сюди зображення плиток маджонгу у форматі PNG.

## Де знайти зображення:

### 1. OpenGameArt.org
- Відвідай: https://opengameart.org/
- Пошук: "mahjong tiles" або "mahjong sprites"
- Безкоштовні ресурси з різними ліцензіями

### 2. Kenney.nl
- Відвідай: https://kenney.nl/assets
- Пошук: "mahjong"
- Безкоштовні набори графіки для ігор

### 3. Itch.io
- Відвідай: https://itch.io/game-assets/free
- Пошук: "mahjong tiles"
- Багато безкоштовних та платних наборів

### 4. GitHub
- Відвідай: https://github.com/search?q=mahjong+tiles+sprites
- Багато open-source проектів з плитками

## Рекомендації

- Розмір зображень: 60x80 пікселів (або більше, гра автоматично масштабує)
- Формат: PNG з прозорим фоном (якщо можливо)
- Якість: чіткі, легко розпізнавані символи

Якщо зображення не знайдено, гра використає placeholder з текстом.
